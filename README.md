# KiCad PCB Projects

This repository contains my PCB design projects created using KiCad.

## Projects

### LED Circuit
Basic LED circuit for learning fundamentals.

### LED with Button
LED controlled using a push button.

### LED with Transistor
Used a transistor to control higher current.

### LED MOSFET Button
MOSFET-based switching circuit.

### LED PWM
PWM signal filtered into analog voltage using RC filter.

## Tools Used
- KiCad
- Tinkercad

## What I Learned
- PCB design basics
- Schematic to layout workflow
- MOSFET vs transistor switching
- PWM and low-pass filters
